Proyectos

Videojuego Nakoa

Un videojuego multiplataforma desarrollado en Unity, disponible para iOS, Android, PC, webGL... Además, los datos se almacenan mediante una API REST en PHP.

Cinema

Desarrollado como parte del curso CS50 de Harvard. El objetivo de este proyecto es crear una aplicación web para la gestión y visualización de información sobre películas. Se usan Python y Django.

Diagnóstico de cáncer con Redes Neuronales

Este TFG se centra en el desarrollo y evaluación de un modelo avanzado de autoencoder para la detección de anomalías en muestras de cáncer de mama. Se usa Python y Keras.

Bot Telegram Finanzas Piso

Un bot de Telegram que automatiza consultas y avisos sobre finanzas del piso compartido. Se conecta mediante una API intermediaria a cualquier banco (Santander, Caixabank,...).

App Repartir Tareas

Aplicación móvil multiplataforma para distribuir y asignar tareas, gestionar pagos, recordatorios y otros de forma colaborativa entre miembros de un equipo. Realizada con React, Expo y Firebase.
